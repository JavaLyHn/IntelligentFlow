# 播客写作技能

## 能力说明
这个技能让你能够：
1. 生成结构化的播客脚本
2. 将文本转换为语音（TTS）
3. 支持多种发音人和语调
4. 输出音频文件和文本稿件

## 使用指南

### 播客脚本生成
1. 确定播客主题和目标受众
2. 设计节目结构（开场、主体、结尾）
3. 生成对话式脚本
4. 标注语气和停顿

### 语音合成
1. 选择发音人
2. 配置语速和音调
3. 调用 TTS 插件生成音频
4. 上传至对象存储返回 URL

## 注意事项
- 播客脚本需口语化，避免书面语
- TTS 文本需注意分片，单次不超过 500 字节
