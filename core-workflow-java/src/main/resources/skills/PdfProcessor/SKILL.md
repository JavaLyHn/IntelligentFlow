# PDF 处理技能

## 能力说明
这个技能让你能够：
1. 提取 PDF 中的文本内容和表单字段
2. 识别 PDF 中的表格、图片和代码块
3. 将 PDF 转换为 Markdown 格式
4. 对 PDF 内容进行智能摘要和分析

## 使用指南

### PDF 文本提取
当用户要求提取 PDF 内容时，使用以下步骤：
1. 首先分析 PDF 结构（页数、是否加密）
2. 调用 OCR 工具逐页识别内容
3. 解析段落、表格、代码块等结构化内容
4. 输出 Markdown 格式文本

### PDF 表单处理
1. 识别所有表单字段
2. 输出字段列表，包含字段名、类型、当前值
3. 支持表单填写和数据导出

### PDF 内容分析
1. 提取全文后进行语义分析
2. 生成文档摘要
3. 识别关键实体和主题

## 注意事项
- 处理大型 PDF 时要注意内存，建议分页处理
- 某些加密 PDF 可能无法处理，需提前告知用户
- OCR 识别精度受图片质量影响
