# 代码审查技能

## 能力说明
这个技能让你能够：
1. 对代码进行自动化审查，识别潜在问题
2. 按照语言规范进行代码格式化
3. 检查代码质量和安全漏洞
4. 生成审查报告和改进建议

## 使用指南

### 代码审查流程
当用户要求审查代码时，使用以下步骤：
1. 首先识别代码语言和框架
2. 分析代码结构和设计模式
3. 检查命名规范、代码风格
4. 识别潜在的安全漏洞和性能问题
5. 输出审查报告，包含问题描述、严重程度和修复建议

### 代码格式化
1. 根据语言选择对应的格式化规则
2. 应用缩进、空格、换行等格式规范
3. 优化 import 排序
4. 输出格式化后的代码

## 注意事项
- 审查时需区分业务逻辑问题和风格问题
- 安全漏洞应标记为高优先级
- 格式化不应改变代码语义
