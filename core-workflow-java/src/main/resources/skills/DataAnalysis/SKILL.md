# 数据分析技能

## 能力说明
这个技能让你能够：
1. 对结构化数据进行统计分析
2. 生成可视化图表（柱状图、折线图、饼图等）
3. 输出数据洞察和趋势分析报告
4. 支持多种数据源（CSV、数据库、API）

## 使用指南

### 数据分析流程
1. 确定数据源和数据格式
2. 执行数据清洗和预处理
3. 选择合适的分析方法（描述统计、趋势分析、对比分析）
4. 生成可视化图表
5. 输出分析结论和建议

### 图表生成
1. 根据数据特征选择图表类型
2. 配置图表参数（标题、坐标轴、图例）
3. 生成 ECharts 配置或图片

## 注意事项
- 大数据集需考虑分页和采样
- 注意数据隐私和脱敏处理
